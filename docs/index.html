Ebony Mono

A lightweight and minimalist classless CSS stylesheet

  • GitHub
  • npm
  • CDN

Overview

A classess CSS stylesheet aiming for a small size, versatility, simplicity, and minimalism. This can be a great starting point for a new website, or the majority of styles, for a simple one.

Ebony Mono is inspired by concrete and magick.

Layout

When used as a direct child of <body>, the <main> element is centered and compacted within a readable width (640px).

Within <main>, add <header>, <footer>, and instances of <section> when presenting long-form content.

Typography

Font family

Font family is a complicated matter. Outside of choosing the fonts themselves, there is no simple and superior solution to hosting fonts and delivering them to users. As such, Ebony Mono opts for system's existing UI fonts, using what the users already had. This can be easily overridden with the developer's informed and considered choice of solution.
